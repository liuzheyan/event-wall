# Event Wall (弹幕墙)

## 📋 功能点说明

### 📱 手机端 (用户端)

**1. 弹幕展示**
- **实时投屏**：屏幕实时展示所有弹幕内容。
- **自我高亮**：用户自己发送的弹幕颜色加深，便于快速识别。
- **状态同步**：当大屏端执行暂停或清屏操作时，手机端会同步收到通知并展示相应提示。

**2. 弹幕发送**
- **发送功能**：支持用户输入并发送弹幕。
- **发送限制**：
  - **频率限制**：每 3 秒仅允许发送一条弹幕，防止刷屏。
  - **字数限制**：单条弹幕长度限制在 30 字以内。
- **内容安全**：集成基础敏感词过滤功能，发送的敏感词将被自动替换为 `***`。

### 💻 PC 端 (大屏/管理端)

**1. 弹幕展示**
- **全量展示**：大屏展示所有用户发送的实时弹幕。
- **空闲填充**：当没有新弹幕时，自动滚动播放默认/预设的弹幕内容，保持氛围。

**2. 播放控制**
- **暂停/继续**：管理员可控制弹幕暂停。该状态会实时同步至手机端，手机端将展示暂停提示。
- **清屏控制**：管理员可一键清空屏幕弹幕。该状态会实时同步至手机端。
